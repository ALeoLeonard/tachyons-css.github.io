# TACHYONS

Documentation for [Tachyons](http://tachyons.io).

## Installation

```sh
git clone https://github.com/tachyons-css/tachyons-css.github.io.git && cd $_
npm i -g superstatic && npm i
npm start
open  http://localhost:3474
```
