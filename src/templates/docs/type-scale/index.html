Often times - websites devote a non-trivial amount of css to setting the font-size of elements. They declare an unnecessary amount of different font-sizes that don't come close to resembling a sane type scale.

To create and design an easily readable interface, you don't need 40 to 50 font-sizes. A simple ratio-based type-scale is the foundation of readable interface design.

There are eight font-sizing classes to choose from. Using the class extension namespaces you can set the font-size for any particular breakpoint that you desire.

Don't spend time constantly overriding font-sizes in your css. If you don't like a default font-size for an element, use the utilities to quickly make the text larger or smaller until it looks just right.
